# SKILLS.md — Alan Watts Archive Capabilities & Integrations

This file catalogs the available skills, architectural patterns, and integrations used within the Alan Watts Archive application.

## Available Modules & Capabilities
1. **Root Resource Directory (`/resources/`)**: 19 categorized foundational entities (Organizations, Archives, Foundations, Books, Lectures, Podcasts, YouTube Channels, Websites, Newsletters, Communities, Scholars, Creators, Courses, Events, Tools, GPTs, Transcripts, Libraries, Research Papers).
2. **Audio Playback & Waveform Simulation**: Interactive lecture seminar playback with play/pause state, progress scrubbing, and speed controls.
3. **AI Tools & RAG Search**: 
   - Ask the Archive (Semantic RAG search across 428 cataloged lectures)
   - Quote Verifier (Attribution cross-referencing utility)
   - Advanced Lecture Finder (Multi-criteria filter)
4. **Theme Toggle Engine**: Seamlessly switches between sophisticated Editorial typography and vibrant Neo-Brutalist styling.
5. **Business & Footer Layer**: Dedicated views and navigation links for Speaking, Services, Partners, Accelerator, Sponsor, Licensing Information, and Contact.
